> Desarrollado con HTML y css Este proyecto es una landing page para el evento TechVision,
  un evento tecnológico que reúne a expertos y entusiastas del desarrollo web y la innovación.
  La página web tiene como objetivo proporcionar información sobre el evento y permitir a los usuarios registrarse.


## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS 

## 💻 Instalación y Uso Local

## 1. Clona este repositorio
```bash
git clone https://github.com/maria-fernanda-coder/techvision-landingpage
```
## 2. Navega al directorio del proyecto
```bash
cd techvision-landingpage
```
## 3. Abre el archivo index.html en tu navegador (O usa un servidor local como Live Server en VS Code)
